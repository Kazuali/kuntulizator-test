Кунтулизатор — исправленная версия

Что исправлено:
1. Кнопка «Обновить» перенесена в верхнее меню и больше не висит отдельной кривой строкой.
2. Техническая надпись «данные берутся из Google Sheets» скрыта. При нормальной работе пользователь её не видит.
3. Стили и скрипты подключены как styles-v2.css / app-v2.js / data-v2.js, чтобы браузер не подхватил старую версию из кэша.
4. Вкладка «Статистика» должна заполняться автоматически после введённых результатов.

Как обновить сайт:
1. Зайти в GitHub → репозиторий kuntulizator.
2. Add file → Upload files.
3. Загрузить файлы из этой папки, не ZIP и не саму папку.
4. Нажать Commit changes.
5. Подождать 1–3 минуты.
6. Открыть сайт так: https://kazuali.github.io/kuntulizator/?v=fix2

Важно:
Старые файлы app.js/styles.css/data.js можно не удалять. Новый index.html использует app-v2.js, styles-v2.css и data-v2.js.
