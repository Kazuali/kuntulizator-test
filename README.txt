Кунтулизатор v23

Что изменено:
- На вкладке «Статистика» добавлена история позиций после каждого завершённого матча.
- Добавлены показатели:
  1. сколько раз участник был лидером;
  2. сколько раз участник был в топ-8;
  3. сколько раз участник был в зоне «Ебездалы».
- В блоке «История позиций» показываются лидеры по этим показателям.
- В таблицу статистики участников добавлены колонки «Лидер», «Топ-8», «Ебездалы».
- Логика считает контрольную точку после каждого матча, у которого введён результат.
- Если несколько участников делят 1 место по очкам, лидерство засчитывается всем.

Сохранены правки v22:
- выбор тура на вкладке «Матчи»;
- выбор тура и матча на вкладке «Прогнозы»;
- фото Месси наверху сайта;
- телевизор с чатом админа;
- мобильные исправления по автозуму;
- сдача прогнозов через Apps Script.
