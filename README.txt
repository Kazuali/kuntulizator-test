Кунтулизатор v5 — тестовая версия с формой сдачи прогнозов

Что добавлено:
1. Вкладка «Сдать прогноз».
2. Прогноз сдаётся только на весь тур целиком.
3. Нельзя отправить, если заполнены не все матчи.
4. Перед отправкой появляется экран проверки.
5. Нужно поставить галочку «Я проверил прогнозы».
6. До дедлайна можно отправить новую версию.
7. После дедлайна форма блокируется.
8. Прогнозы скрыты до дедлайна. До дедлайна видно только «сдал / не сдал».

Что загрузить в GitHub test-репозиторий:
- index.html
- styles-v5.css
- data-v5.js
- app-v5.js
- README.txt
- kuntulizator_data.json

Важно:
Форма отправки заработает только после настройки Google Apps Script.
После создания Web App URL нужно вставить ссылку в data-v5.js:
submissionWebAppUrl: "PASTE_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE"

Для теста дедлайн стоит на 2026-12-31 23:59 МСК, чтобы форма была открыта.
Перед реальным запуском дедлайн нужно заменить на время начала первого матча тура и в data-v5.js, и в Google Apps Script.
